## How to Contribute

1. **Bug reports** — [Open an issue](https://github.com/Nebukam/PCGExElementsZoneGraph/issues) with as much detail as possible: steps to reproduce, engine version, stack traces, node setup, etc.
2. **Feature requests** — Start a discussion on the [Discord server](https://discord.gg/mde2vC5gbE) or open an issue.
3. **Pull requests** — Fork the repo, make your changes on a branch, and submit a PR. Keep PRs focused and small when possible.

---

## Guidelines

- Follow the existing code style and conventions.
- Test your changes against the supported engine versions (5.7+).
- Make sure both **PCGExtendedToolkit** and **ZoneGraph** are enabled when testing.
- Keep commits clean and descriptive.
